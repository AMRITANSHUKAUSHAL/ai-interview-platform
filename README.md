# 🚀 AI Interview Arena

An advanced AI-powered mock interview platform built using React, Node.js, PostgreSQL, Prisma, and AI evaluation.

Users can attend technical interviews, answer questions using text or voice, receive instant AI feedback, track interview history, and simulate real-world interview experiences.

# ⚙ Installation

## Clone Repository

```bash
git clone <your-repo-url>
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## Prisma Setup

```bash
npx prisma migrate dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server folder:

```env
DATABASE_URL=your_database_url

JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_openai_key
```
